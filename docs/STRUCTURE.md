# Project Structure

```
bigcommerce-coupon-manager/
├── src/
│   ├── server.js                 # Express server & API routes
│   └── utils/
│       ├── api.js               # BigCommerce API functions
│       ├── coupon.js            # Coupon generation utilities
│       └── date.js              # Date conversion utilities
│
├── public/
│   └── index.html               # Web UI for coupon management
│
├── scripts/
│   ├── bulk-generate-coupons.js # Generate bulk coupons from CLI
│   ├── export-all-coupons.js    # Export all store coupons to CSV
│   ├── create-coupon.js         # Create single coupons interactively
│   └── fetch-coupon-data.js     # Fetch coupon data from Excel file
│
├── config/
│   ├── .env                     # Environment variables (DO NOT COMMIT)
│   ├── .env.example             # Template for .env file
│   └── coupons.json             # Coupon configuration template
│
├── data/
│   └── (Generated exports)      # CSV and XLSX files generated by scripts
│
├── docs/
│   ├── README.md                # Project documentation
│   └── QUICKSTART.md            # Quick start guide
│
├── package.json                 # Project dependencies & scripts
├── .gitignore                   # Git ignore rules
└── node_modules/                # Dependencies (not in repo)
```

## File Descriptions

### /src
- **server.js** - Main Express server handling API routes for the web UI
- **utils/api.js** - Reusable functions for BigCommerce API calls
- **utils/coupon.js** - Coupon code/name generation logic
- **utils/date.js** - Date format conversion utilities

### /public
- **index.html** - Web interface for managing coupons with product selection

### /scripts
CLI scripts for bulk operations:
- **bulk-generate-coupons.js** - Generate 100-800 coupons with custom prefixes
- **export-all-coupons.js** - Export all store coupons with timestamps
- **create-coupon.js** - Interactive single coupon creation
- **fetch-coupon-data.js** - Enrich Excel coupon list with BigCommerce data

### /config
- **.env** - Your BigCommerce credentials (created from .env.example)
- **.env.example** - Template with required environment variables
- **coupons.json** - Example coupon configuration

### /data
Output directory for generated files:
- Timestamped CSV exports from bulk generation
- Timestamped XLSX files with coupon details

### /docs
Documentation files:
- **README.md** - Full project documentation
- **QUICKSTART.md** - Get started quickly
